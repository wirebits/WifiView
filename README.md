# WifiView
A tool which show the all wifi interfaces (on-board and adapters) with some basic details.

<h1>Tested Systems</h1>
The tool is currently tested on : <br>
1. Raspberry Pi OS<br>
2. Kali Linux<br>
The testing is going on different systems.

<h1>Install and Run</h1>
1. Download or Clone the Repository.<br>
2. Open the folder and type the following command :<br><br>

```
make
```
<h1>Key Features</h1>
- Simple and clean interface.<br>
- It shows : Wifi Interface , Driver of the wireless adapter , Mode and Internet Access.
